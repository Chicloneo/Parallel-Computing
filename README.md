# Parallel-Computing
This is an Introduction to Parallel Computing with Python. On each project, I will explain a different concept: multiprocessing, threads, locks, sockets, ... 

I will add a Markdown document with a more detailed explanation on each folder, even though the code will have comments. You can see this by opening the README.md file. You may find some comments in spanish and others in english within the code.

You probably need to install the Quarto extension on VSCode to visualize (render) the .qmd file. RStudio can do this too. But opening the .md will work well.
