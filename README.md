# Parallel-Computing
This is an Introduction to Parallel Computing with Python. On each project, I will explain a different concept: multiprocessing, threads, locks, sockets, ... 

I will add a Markdown document with a more detailed explanation, even though the code will have comments. You can see thiss by opening the .html file. You may find some comments in spanish and others in english within the code.

You probably need to install the Quarto extension on VSCode to visualize the .qmd file. This file is the heart of the .html, but you will usually not need to open it. RStudio can do this too.
